# Aziz Honey Flutter UI

واجهة Flutter مبدئية (UI only) مستوحاة من تصميم:
https://github.com/abuanwar072/E-commerce-App-UI-Flutter

## الحالة الحالية
- UI كامل مبدئي بدون Backend
- بدون لوحة تحكم
- بدون قاعدة بيانات
- ألوان وهوية قريبة من Aziz Honey
- RTL عربي

## الشاشات
- الرئيسية
- المتجر (التصنيفات + المنتجات)
- تفاصيل المنتج
- السلة
- إتمام الطلب
- تسجيل الدخول / إنشاء حساب
- حسابي
- طلباتي + تتبع الطلب
- المفضلة
- تواصل معنا
- من نحن
- الأسئلة الشائعة
- شهادة مختبر الجودة
- سياسة الإرجاع والتعويض

## تشغيل
```bash
flutter pub get
flutter run
```

## إضافات UX المنفذة
- Splash Screen براندد بهوية Aziz Honey
- Drawer مبدئي للتنقل
- Animated transitions بين تبويبات التطبيق

## الخطوة التالية
- ربط WooCommerce API
- Auth / Cart / Checkout الحقيقي
- Firebase Notifications
